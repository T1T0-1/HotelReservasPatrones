# Guía de video demo (5-8 minutos)

1. Mostrar el proyecto abierto en NetBeans y el enlace de GitHub (30 s).
2. Explicar el problema y la arquitectura por capas (45 s).
3. Ejecutar: listar habitaciones y registrar una nueva (45 s).
4. Crear una reserva para la habitación 101 (1 min).
5. Mostrar que la habitación cambia a RESERVADA y que el CSV se actualiza (45 s).
6. Finalizar o cancelar la reserva y comprobar que la habitación queda DISPONIBLE (45 s).
7. Generar el reporte y abrir `data/auditoria.log` (40 s).
8. Explicar 6 patrones, SOLID y GRASP con la tabla del informe (1-2 min).
9. Ejecutar `ant smoke` y, con internet, `ant test` (30 s).

El video debe ser grabado y su enlace añadido al README y al informe antes de entregar.
